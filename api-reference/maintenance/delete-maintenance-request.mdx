---
title: 'Delete Maintenance Request'
api: 'DELETE /maintenance-requests/{id}'
description: 'Delete a maintenance request'
---

## Request

Delete a maintenance request by ID.

<ParamField path="id" type="integer" required>
  Maintenance request ID
</ParamField>

## Response

Returns a success message.

<ResponseField name="message" type="string">
  Success message
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X DELETE https://office.homestayos.com/api/maintenance-requests/1 \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/maintenance-requests/1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://office.homestayos.com/api/maintenance-requests/1',
    headers={
        'Authorization': 'Bearer {your_token}',
    }
)

data = response.json()
```

</CodeGroup>

## Example Response

{
  "message": "Maintenance request deleted successfully"
