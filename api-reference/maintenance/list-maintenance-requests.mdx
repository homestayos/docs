api: 'GET /maintenance-requests'
description: 'Get all maintenance requests with optional filtering'
---

## Request

Retrieve a paginated list of maintenance requests for the authenticated tenant with optional filtering.

<ParamField body="status" type="string" enum={['pending', 'in_progress', 'completed', 'cancelled']}>
  Filter by status
</ParamField>

<ParamField body="priority" type="string" enum={['low', 'normal', 'high', 'urgent']}>
  Filter by priority
</ParamField>

<ParamField body="category" type="string">
  Filter by category
</ParamField>

<ParamField body="property_id" type="integer">
  Filter by property ID
</ParamField>

<ParamField body="assigned_to_id" type="integer">
  Filter by assigned user ID
</ParamField>

<ParamField body="search" type="string">
  Search in title and description
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  Sort field (e.g., created_at, scheduled_date)
</ParamField>

<ParamField body="sort_order" type="string" default="desc" enum={['asc', 'desc']}>
  Sort order
</ParamField>

<ParamField body="per_page" type="integer" default={15}>
  Number of items per page
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET http://localhost:8000/api/maintenance-requests \
  -H "Authorization: Bearer {your_token}" \
  -H "Content-Type: application/json" \
  -G \
  -d "status=pending" \
  -d "priority=high" \
  -d "per_page=20"
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/maintenance-requests?status=pending&priority=high&per_page=20', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer {your_token}',
    'Content-Type': 'application/json',
  },
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/maintenance-requests',
    headers={
        'Authorization': 'Bearer {your_token}',
        'Content-Type': 'application/json',
    },
    params={
        'status': 'pending',
        'priority': 'high',
        'per_page': 20,
    }
)

data = response.json()
```

</CodeGroup>
