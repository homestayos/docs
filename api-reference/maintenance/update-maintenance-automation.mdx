---
title: 'Update Maintenance Automation'
api: 'PUT /maintenance-automations/{id}'
description: 'Update an existing maintenance automation'
---

## Request

Update a maintenance automation. All fields are optional - only include the fields you want to update.

<ParamField path="id" type="integer" required>
  Maintenance automation ID
</ParamField>

<ParamField body="name" type="string">
  Automation name
</ParamField>

<ParamField body="description" type="string">
  Automation description
</ParamField>

<ParamField body="property_id" type="integer">
  Associated property ID
</ParamField>

<ParamField body="trigger" type="string" enum={['after_booking_confirmed', 'before_checkin', 'after_checkin', 'before_checkout', 'after_checkout', 'guest_message', 'scheduled_maintenance', 'inspection_due', 'property_age_days', 'booking_count']}>
  Trigger type
</ParamField>

<ParamField body="is_active" type="boolean">
  Whether the automation is active
</ParamField>

## Response

Returns the updated maintenance automation.

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="automation" type="object">
  The updated maintenance automation object
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PUT http://localhost:8000/api/maintenance-automations/1 \
  -H "Authorization: Bearer {your_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "is_active": false
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/maintenance-automations/1', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer {your_token}',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    is_active: false,
  }),
});

const data = await response.json();
```

```python Python
import requests

response = requests.put(
    'http://localhost:8000/api/maintenance-automations/1',
    headers={
        'Authorization': 'Bearer {your_token}',
        'Content-Type': 'application/json',
    },
    json={
        'is_active': False,
    }
)

data = response.json()
```

</CodeGroup>
