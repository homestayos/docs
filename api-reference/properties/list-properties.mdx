---
title: 'List Properties'
api: 'GET /properties'
description: 'Get all properties for the authenticated tenant with optional filtering'
---

## Request

Retrieve a list of properties with optional filtering by status and search query.

<ParamField query name="status" type="string">
  Filter by property status. Options: `active`, `inactive`
</ParamField>

<ParamField query name="search" type="string">
  Search properties by name, address, or description
</ParamField>

<ParamField query name="per_page" type="integer" default="15">
  Number of results per page
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of property objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8000/api/properties?status=active&per_page=20" \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/properties?status=active&per_page=20', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/properties',
    params={'status': 'active', 'per_page': 20},
    headers={'Authorization': 'Bearer {your_token}'}
)

data = response.json()
```
