---
title: 'Get Property'
api: 'GET /properties/{id}'
description: 'Get a specific property by ID'
---

## Request

Retrieve detailed information about a specific property.

<ParamField path name="id" type="integer" required>
  Property ID
</ParamField>

## Response

Returns the property object with all details.

## Example Request

<CodeGroup>

```bash cURL
curl -X GET http://localhost:8000/api/properties/1 \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/properties/1', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const property = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/properties/1',
    headers={'Authorization': 'Bearer {your_token}'}
)

property = response.json()
```

</CodeGroup>
