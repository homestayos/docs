---
title: 'Update Booking'
api: 'PUT /bookings/{id}'
description: 'Update an existing booking'
---

## Request

Update booking information. Only include fields you want to update.

<ParamField path name="id" type="integer" required>
  Booking ID
</ParamField>

<ParamField body name="check_in_date" type="string" format="date">
  Check-in date (YYYY-MM-DD)
</ParamField>

<ParamField body name="check_out_date" type="string" format="date">
  Check-out date (YYYY-MM-DD)
</ParamField>

<ParamField body name="guests_count" type="integer">
  Number of guests
</ParamField>

<ParamField body name="nightly_rate" type="number">
  Nightly rate
</ParamField>

<ParamField body name="status" type="string">
  Booking status. Options: `pending`, `confirmed`, `checked_in`, `checked_out`, `cancelled`
</ParamField>

<ParamField body name="special_requests" type="string">
  Special requests or notes
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PUT https://office.homestayos.com/api/bookings/1 \
  -H "Authorization: Bearer {your_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "confirmed",
    "special_requests": "Late check-in requested"
  }'
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/bookings/1', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer {your_token}',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    status: 'confirmed',
    special_requests: 'Late check-in requested',
  }),
});

const booking = await response.json();
```

```python Python
import requests

response = requests.put(
    'https://office.homestayos.com/api/bookings/1',
    headers={
        'Authorization': 'Bearer {your_token}',
        'Content-Type': 'application/json',
    },
    json={
        'status': 'confirmed',
        'special_requests': 'Late check-in requested',
    }
)

booking = response.json()
```

</CodeGroup>
