---
title: 'Cancel Booking'
api: 'POST /bookings/{id}/cancel'
description: 'Cancel a booking'
---

## Request

Cancel an existing booking. This will change the booking status to `cancelled`.

<ParamField path name="id" type="integer" required>
  Booking ID to cancel
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST http://localhost:8000/api/bookings/1/cancel \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/bookings/1/cancel', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const booking = await response.json();
```

```python Python
import requests

response = requests.post(
    'http://localhost:8000/api/bookings/1/cancel',
    headers={'Authorization': 'Bearer {your_token}'}
)

booking = response.json()
```

</CodeGroup>
