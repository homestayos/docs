---

## Request

Retrieve a list of bookings with optional filtering by status, property, date range, and more.

<ParamField query name="status" type="string">
  Filter by booking status. Options: `pending`, `confirmed`, `checked_in`, `checked_out`, `cancelled`
</ParamField>

<ParamField query name="property_id" type="integer">
  Filter bookings by property ID
</ParamField>

<ParamField query name="start_date" type="string" format="date">
  Filter bookings starting from this date (YYYY-MM-DD)
</ParamField>

<ParamField query name="end_date" type="string" format="date">
  Filter bookings ending before this date (YYYY-MM-DD)
</ParamField>

<ParamField query name="filter" type="string">
  Quick filter. Options: `upcoming`, `past`, `active`
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of booking objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8000/api/bookings?status=confirmed&filter=upcoming" \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:8000/api/bookings?status=confirmed&filter=upcoming',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {your_token}',
    },
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/bookings',
    params={'status': 'confirmed', 'filter': 'upcoming'},
    headers={'Authorization': 'Bearer {your_token}'}
)

data = response.json()
```
