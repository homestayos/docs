## Request

Retrieve a list of conversations/messages with optional filtering.

<ParamField query name="status" type="string">
  Filter conversations by status
</ParamField>

<ParamField query name="per_page" type="integer" default="15">
  Number of results per page
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://office.homestayos.com/api/conversations?status=open&per_page=20" \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch(
  'https://office.homestayos.com/api/conversations?status=open&per_page=20',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {your_token}',
    },
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://office.homestayos.com/api/conversations',
    params={'status': 'open', 'per_page': 20},
    headers={'Authorization': 'Bearer {your_token}'}
)

data = response.json()
```
