---
title: 'Get Revenue Analytics'
api: 'GET /analytics/revenue'
description: 'Get revenue analytics with date range filtering'
---

## Request

Retrieve revenue analytics and financial data for your properties.

<ParamField query name="start_date" type="string" format="date">
  Start date for the analytics period (YYYY-MM-DD)
</ParamField>

<ParamField query name="end_date" type="string" format="date">
  End date for the analytics period (YYYY-MM-DD)
</ParamField>

<ParamField query name="property_id" type="integer">
  Filter analytics by specific property ID
</ParamField>

## Response

<ResponseField name="total_revenue" type="number">
  Total revenue across all time
</ResponseField>

<ResponseField name="period_revenue" type="number">
  Revenue for the specified date range
</ResponseField>

<ResponseField name="by_property" type="array">
  Revenue breakdown by property
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://office.homestayos.com/api/analytics/revenue?start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch(
  'https://office.homestayos.com/api/analytics/revenue?start_date=2024-01-01&end_date=2024-01-31',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {your_token}',
    },
  }
);

const analytics = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://office.homestayos.com/api/analytics/revenue',
    params={'start_date': '2024-01-01', 'end_date': '2024-01-31'},
    headers={'Authorization': 'Bearer {your_token}'}
)

analytics = response.json()
```

</CodeGroup>
