## Request

Retrieve a list of tasks with optional filtering by status and property.

<ParamField query name="status" type="string">
  Filter by task status. Options: `pending`, `in_progress`, `completed`, `cancelled`
</ParamField>

<ParamField query name="property_id" type="integer">
  Filter tasks by property ID
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://office.homestayos.com/api/tasks?status=pending&property_id=1" \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch(
  'https://office.homestayos.com/api/tasks?status=pending&property_id=1',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {your_token}',
    },
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://office.homestayos.com/api/tasks',
    params={'status': 'pending', 'property_id': 1},
    headers={'Authorization': 'Bearer {your_token}'}
)

data = response.json()
```
