---
title: 'Get Calendar Sync URL'
api: 'GET /properties/{property}/calendar-url'
description: 'Get iCal calendar URL for syncing with external calendar applications'
---

## Request

Retrieve an iCal calendar URL for a property that can be used to sync availability and bookings with external calendar applications like Google Calendar, Outlook, or Apple Calendar.

<ParamField path name="property" type="integer" required>
  Property ID
</ParamField>

## Response

<ResponseField name="url" type="string" format="uri">
  iCal calendar URL that can be subscribed to in external calendar applications
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET http://localhost:8000/api/properties/1/calendar-url \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/properties/1/calendar-url', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/properties/1/calendar-url',
    headers={'Authorization': 'Bearer {your_token}'}
)

data = response.json()
```

</CodeGroup>

## Example Response

```json
{
  "url": "http://localhost:8000/api/properties/1/calendar.ics?token=abc123"
}
```

<Tip>
  You can subscribe to this URL in Google Calendar, Outlook, Apple Calendar, or any other calendar application that supports iCal feeds. The calendar will automatically update when bookings change.
</Tip>
