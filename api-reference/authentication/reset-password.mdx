---
title: 'Reset Password'
api: 'POST /reset-password'
description: 'Reset password using a reset token from email'
---

## Request

Reset a user's password using a reset token received via email.

<ParamField body="email" type="string" format="email" required>
  User's email address
</ParamField>

<ParamField body="token" type="string" required>
  Password reset token from the email
</ParamField>

<ParamField body="password" type="string" format="password" required>
  New password (minimum 8 characters)
</ParamField>

<ParamField body="password_confirmation" type="string" format="password" required>
  Password confirmation (must match password)
</ParamField>

## Response

Returns a success message.

<ResponseField name="message" type="string">
  Success message
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://office.homestayos.com/api/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "token": "reset_token_from_email",
    "password": "newpassword123",
    "password_confirmation": "newpassword123"
  }'
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/reset-password', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'user@example.com',
    token: 'reset_token_from_email',
    password: 'newpassword123',
    password_confirmation: 'newpassword123',
  }),
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://office.homestayos.com/api/reset-password',
    headers={
        'Content-Type': 'application/json',
    },
    json={
        'email': 'user@example.com',
        'token': 'reset_token_from_email',
        'password': 'newpassword123',
        'password_confirmation': 'newpassword123',
    }
)

data = response.json()
```

</CodeGroup>
