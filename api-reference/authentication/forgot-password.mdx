---
title: 'Forgot Password'
api: 'POST /forgot-password'
description: 'Request a password reset email'
---

## Request

Request a password reset email to be sent to the user's email address.

<ParamField body="email" type="string" format="email" required>
  User's email address
</ParamField>

## Response

Returns a success message (email is sent if the email exists in the system).

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<Note>
  For security reasons, the API will always return a success message even if the email doesn't exist in the system. This prevents email enumeration attacks.
</Note>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://office.homestayos.com/api/forgot-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/forgot-password', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'user@example.com',
  }),
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://office.homestayos.com/api/forgot-password',
    headers={
        'Content-Type': 'application/json',
    },
    json={
        'email': 'user@example.com',
    }
)

data = response.json()
```

</CodeGroup>

## Example Response

```json
{
  "message": "Password reset email sent if the email exists in our system"
}
```
