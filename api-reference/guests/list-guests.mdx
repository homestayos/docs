## Request

Retrieve a list of guests with optional search filtering.

<ParamField query name="search" type="string">
  Search guests by name, email, or phone number
</ParamField>

<ParamField query name="per_page" type="integer" default="15">
  Number of results per page
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8000/api/guests?search=john&per_page=20" \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:8000/api/guests?search=john&per_page=20',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {your_token}',
    },
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/guests',
    params={'search': 'john', 'per_page': 20},
    headers={'Authorization': 'Bearer {your_token}'}
)

data = response.json()
```
