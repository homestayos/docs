## Request

Create a new guest profile in your system.

<ParamField body name="first_name" type="string" required>
  Guest's first name
</ParamField>

<ParamField body name="last_name" type="string" required>
  Guest's last name
</ParamField>

<ParamField body name="email" type="string" format="email" required>
  Guest's email address
</ParamField>

<ParamField body name="phone" type="string">
  Guest's phone number
</ParamField>

<ParamField body name="address" type="string">
  Guest's street address
</ParamField>

<ParamField body name="city" type="string">
  Guest's city
</ParamField>

<ParamField body name="country" type="string">
  Guest's country
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://office.homestayos.com/api/guests \
  -H "Authorization: Bearer {your_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "first_name": "Jane",
    "last_name": "Smith",
    "email": "jane@example.com",
    "phone": "+1234567890",
    "address": "456 Oak Ave",
    "city": "Los Angeles",
    "country": "USA"
  }'
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/guests', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer {your_token}',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    first_name: 'Jane',
    last_name: 'Smith',
    email: 'jane@example.com',
    phone: '+1234567890',
    address: '456 Oak Ave',
    city: 'Los Angeles',
    country: 'USA',
  }),
});

const guest = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://office.homestayos.com/api/guests',
    headers={
        'Authorization': 'Bearer {your_token}',
        'Content-Type': 'application/json',
    },
    json={
        'first_name': 'Jane',
        'last_name': 'Smith',
        'email': 'jane@example.com',
        'phone': '+1234567890',
        'address': '456 Oak Ave',
        'city': 'Los Angeles',
        'country': 'USA',
    }
)

guest = response.json()
```
