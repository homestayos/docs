---
title: 'Get Guest Statistics'
api: 'GET /guests/stats'
description: 'Get statistics about guests'
---

## Request

Retrieve aggregated statistics about guests in your account.

## Response

<ResponseField name="total_guests" type="integer">
  Total number of guests
</ResponseField>

<ResponseField name="new_this_month" type="integer">
  Number of new guests added this month
</ResponseField>

<ResponseField name="repeat_guests" type="integer">
  Number of repeat guests (guests with multiple bookings)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET http://localhost:8000/api/guests/stats \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/api/guests/stats', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const stats = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/api/guests/stats',
    headers={'Authorization': 'Bearer {your_token}'}
)

stats = response.json()
```

</CodeGroup>

## Example Response

```json
{
  "total_guests": 150,
  "new_this_month": 25,
  "repeat_guests": 45
}
```
