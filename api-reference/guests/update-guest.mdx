## Request

Update guest information. Only include fields you want to update.

<ParamField path name="id" type="integer" required>
  Guest ID
</ParamField>

<ParamField body name="first_name" type="string">
  Guest's first name
</ParamField>

<ParamField body name="last_name" type="string">
  Guest's last name
</ParamField>

<ParamField body name="email" type="string" format="email">
  Guest's email address
</ParamField>

<ParamField body name="phone" type="string">
  Guest's phone number
</ParamField>

<ParamField body name="address" type="string">
  Guest's street address
</ParamField>

<ParamField body name="city" type="string">
  Guest's city
</ParamField>

<ParamField body name="country" type="string">
  Guest's country
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PUT https://office.homestayos.com/api/guests/1 \
  -H "Authorization: Bearer {your_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "+1987654321",
    "address": "789 Pine St"
  }'
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/guests/1', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer {your_token}',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    phone: '+1987654321',
    address: '789 Pine St',
  }),
});

const guest = await response.json();
```

```python Python
import requests

response = requests.put(
    'https://office.homestayos.com/api/guests/1',
    headers={
        'Authorization': 'Bearer {your_token}',
        'Content-Type': 'application/json',
    },
    json={
        'phone': '+1987654321',
        'address': '789 Pine St',
    }
)

guest = response.json()
```
