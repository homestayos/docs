## Request

Retrieve detailed information about a specific guest.

<ParamField path name="id" type="integer" required>
  Guest ID
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://office.homestayos.com/api/guests/1 \
  -H "Authorization: Bearer {your_token}"
```

```javascript JavaScript
const response = await fetch('https://office.homestayos.com/api/guests/1', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer {your_token}',
  },
});

const guest = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://office.homestayos.com/api/guests/1',
    headers={'Authorization': 'Bearer {your_token}'}
)

guest = response.json()
```

</CodeGroup>
